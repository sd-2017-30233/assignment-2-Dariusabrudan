In order to use the book store application, you have to install a IDE(Integrated development environment) for Java.
In order to access the application, you need to run the Controller, which has the main method. A view with the log in window
will appear. There you need to enter the username and password and click OK in order to log in.
You can log in with a user that exists in the user.xml file, eighter an employee or the administrator.
If you log in as an employee, you have a set of operations and if you log in as administrator, you have other operations available, 
listed for an easy access.
